RSA BASED ON RUBY!
===================



----------


Getting started
-------------

#### 1. Clone this project
```
$ git clone https://github.com/soundsnick/rsa-ruby
```

#### 2. Generate keys
```
$ cd rsa-ruby
```
> Uncomment last line of the file generate_keys.rb
```
$ ruby server.rb
```
> Comment back last line of the file generate_keys.rb

#### 3. Run server

```
$ ruby server.rb
```

#### 4. Go to localhost:8888
PROFIT!!!
